Owner indicator options (instead of crown emoji)

A — Pulsing border only (recommended)

TM
Animated glow ring in user's color
No emoji — pure CSS, renders identically everywhere

+ Cross-platform consistent

+ Scales to any avatar size

+ Color matches the user's theme color

+ Subtle enough to not distract

B — Pulsing border + ⌨️ badge

TM
⌨️
Animated ring + small keyboard badge
Badge below avatar, like a status icon

+ Unmistakable — two signals

+ Badge visible even if animation subtle

- Badge adds visual weight

- ⌨️ emoji renders differently per OS

C — Solid ring + dot indicator

TM
Thick white/bright ring + green "active" dot
Green dot = "has control", same position as role dot

+ No emoji at all

+ Green dot is universally "active/live"

- White ring might clash with light themes

- Could confuse with "selected" state

D — Glow ring + status bar only

TM
Subtle animated ring on avatar
Primary indicator is the status bar badge

+ Cleanest avatar area

+ Status bar is always visible

- Easy to miss which avatar is the owner

- Need to read the bar to know who controls
