Hijack Mode (shown above)

Owner gets 3 indicators:
👑 Crown above avatar
Pulsing glow border on avatar
⌨️ Tim controlling badge in status bar
Owner's pin label: shows ⌨️ instead of 📌, thicker edge dot with glow. Unmistakable.
Non-owners: can view, select text, pin lines — but can't type. Their indicators are normal (no crown, no pulse, no ⌨️).

Open Mode (alternative)

No single owner. Multiple operators/admins can type simultaneously.
Each active typer gets a pulsing dot on their avatar (no crown). Status bar shows "Jane, Tim typing" instead of "controlling".
Edge bars show a pulsing segment at cursor position for each typer. No special ⌨️ treatment since no exclusive owner.
